Main Page

This is where you begin the program. All functions accessed via buttons can also be accessed through the 'File' menu bar.

Play Game

Once you've selected a game from the 'Select a Game' list, click here to begin selected game. For more information, check the Play Game page.

New Game

Click here to create a new game from scratch. For more information, check the New Game page.

Edit Game

Once you've selected a game from the 'Select a Game' list, click here to edit selected game's questions. For more information, check the Edit Game page.

Delete Game

Once you've selected a game from the 'Select a Game' list, click here to delete selected game. For more information, check the Delete Game page.

Import From File

Click here to access File Explorer and select game file to import into Jeopardy game database. For more information, check the Import From File page.

Export From File

Once you've selected a game from the 'Select a Game' list, click here to access File Explorer and select location to export selected game to. For more information, check the Export From File page.

Click here to return to Main Page.
