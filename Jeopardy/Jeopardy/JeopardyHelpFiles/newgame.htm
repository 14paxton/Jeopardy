New Game

Here you can create a new Jeopardy game. You will begin with providing the game's name, question time limit, number of categories, and number of questions per category.

Game Name

Game name can be no more than 50 characters long.

Question Time Limit

Time limit is chosen via a drop-down list. Default is '1 minute'; other options include '30 seconds', '90 seconds', '2 minutes', and '3 minutes'.

Number of Categories

Range for number of categories is 3-8, with 6 being the default choice.

Questions Per Category

Range for number of questions per category is 3-8, with 5 being the default choice.

All settings can be changed after initial creation. Press 'Cancel' to cancel creation of game. Press 'Create New Game' to create game template. Game will be saved into Jeopardy database. For more information on naming categories and creating questions, check the Edit Game page.
