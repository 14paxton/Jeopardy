Troubleshooting

Having trouble with the program? You can try these troubleshooting methods by accessing the Troubleshooting page via the Help menu.

Install Access Runtime

The Access Database Engine is a required component for Jeopardy to run corectly. If you are getting database errors when opening the program or attempting to perform simple actions then please click the button below to launch the Access installer.

Compact and Repair

If the program is running slowly then use the below option to Compact and Repair the Access database that this program uses. Alternatively, if you have Microsoft Access installed, you could open up the database in Access and Compact and Repair from there.

Restore To Default

If your current games database is corrupt and giving errors during runtime, consider restoring it to default. Warning. This means that you will lose any games that you made, but the games that came pre-installed with this game will be restored. This is like a factory reset!
